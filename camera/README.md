# Camera stream Publisher

This project is intended to publish camera frames asynchronously using UDP protocol. 
Camera stream must be shown asynchronously in addition to publishing. 
# Dependencies

 - OpenCV=4.5.2
 - Boost ASIO=1.83.0
 - cxxopts

## Directory structure

 - include
	 - options.hpp
 - src
	 - publisher.cpp
	 - subscriber.cpp
- CMakeLists.txt 
- README.md
